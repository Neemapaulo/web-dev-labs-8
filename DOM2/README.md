# Student Grade Calculator Web Application

## Objective

To develop a dynamic web application that captures user input, validates scores, calculates grades using JavaScript, and displays results instantly through DOM manipulation without page reload.

## What Was Done

- Created a user-friendly interface using HTML and CSS.
- Added a score input field and Calculate Grade button.
- Implemented JavaScript event handling and input validation.
- Calculated grades based on score ranges (A–F).
- Displayed results dynamically using DOM manipulation.
- Added automatic input reset after each calculation.

## Challenges Faced

- Linking HTML, CSS, and JavaScript files correctly.
- Handling invalid or empty user inputs.
- Ensuring grades were calculated accurately according to the given ranges.
- Updating results dynamically without refreshing the page.
