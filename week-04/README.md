#  WEEK 4: Understanding Layout Control in CSS

# Objective

The objective of this lab was to develop understanding and practical skills in controlling webpage layout using CSS concepts such as:

- Box Model
  - Margin
  - Padding
  - Border
- Positioning
  - Static
  - Relative
  - Absolute
  - Fixed
  - Sticky
- Float and Clear

The lab also aimed to demonstrate how layout behavior changes when these concepts are applied incorrectly or correctly.


# What Was Done

The following tasks were completed during the lab:

## 1. Box Model Demonstration
Created multiple boxes using `<div>` elements that contained headings and paragraphs.

Applied:
- Padding to create internal spacing
- Borders to create visible boundaries
- Margins to create space between boxes

This demonstrated how the CSS box model controls spacing and element appearance.


## 2. Positioning Demonstration

Created:
- A fixed header
- A sticky navigation bar
- A container with a positioned label

Applied:
- `position: relative` on the container
- `position: absolute` on the label
- `position: fixed` on the header
- `position: sticky` on the navigation bar

This demonstrated how elements behave differently when removed from the normal document flow.


## 3. Float and Clear Layout

Created:
- A main content section
- A sidebar
- A footer

Applied:
- `float: left` to the main content
- `float: right` to the sidebar
- `clear: both` to the footer

This demonstrated how float affects layout structure and why clearing is important.

---

## 4. Integrated Layout

Combined all concepts into a single webpage layout including:
- Fixed header
- Sticky navigation
- Two-column floated layout
- Styled content boxes
- Absolute positioned labels
- Cleared footer

The lab was completed without using:
- Flexbox
- CSS Grid

# Challenges Faced

Several challenges were encountered during the lab:

## 1. Linking HTML and CSS
Initially, the CSS styles did not apply because the HTML file was not correctly linked to the CSS file.

## 2. Understanding Margin vs Padding
It was initially difficult to visually distinguish between margin and padding.


## 3. Fixed Header Overlapping Content
The fixed header covered part of the webpage content during scrolling.
