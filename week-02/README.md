# WEEK 2 LAB ACTIVITY

## Objective
The objective of this lab was to create a 3-page website demonstrating the use of lists, tables, and forms in HTML.

## What Was Done
- Created a Home page with ordered and unordered lists
- Created a Timetable page using tables with rowspan and colspan
- Created a Contact page with forms including a registration form inside a table

## Challenges Faced
- Understanding how to use rowspan and colspan
- Designing forms inside tables
